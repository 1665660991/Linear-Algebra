# 🚀 快速开始指南

## 30秒启动应用

### macOS / Linux
```bash
./start_linear_algebra.sh
```

### Windows
双击 `linear-algebra-calculator.html` 文件

---

## 📖 5分钟上手教程

### 1️⃣ 矩阵计算器（最常用）

**步骤：**
1. 点击左侧 "🔢 矩阵计算器"
2. 选择矩阵维度（例如 3×3）
3. 点击 "更新矩阵"
4. 在网格中输入数值
5. 选择运算类型（行列式、逆矩阵等）
6. 点击 "计算" 按钮
7. 查看结果和公式

**示例：计算 2×2 行列式**
```
输入矩阵:
[1, 2]
[3, 4]

选择: 行列式 det(A)
点击: 计算

结果: det(A) = -2
```

---

### 2️⃣ 行列式专题

**特色功能：逐步展示**
1. 切换到 "📊 行列式专题"
2. 输入 3×3 矩阵
3. 点击 "显示计算步骤"
4. 查看 Laplace 展开过程

**性质演示：**
- 点击 "交换两行（变号）" → 观察行列式符号变化
- 点击 "某行乘k" → 观察行列式倍数变化
- 点击 "倍加变换" → 观察行列式不变

---

### 3️⃣ 方程组求解

**求解线性方程组：**
```
2x + y - z = 8
-3x - y + 2z = -11
-2x + y + 2z = -3
```

**操作：**
1. 选择 "✏️ 方程组求解"
2. 设置方程个数: 3元
3. 输入增广矩阵:
   ```
   [2, 1, -1, 8]
   [-3, -1, 2, -11]
   [-2, 1, 2, -3]
   ```
4. 点击 "高斯消元法"
5. 查看解: x=2, y=3, z=-1
6. 查看详细步骤

---

### 4️⃣ 向量可视化

**交互式演示：**
1. 切换到 "📈 向量可视化"
2. 输入向量 v₁ = (2, 1)
3. 输入向量 v₂ = (1, 3)
4. 拖动滑块调整系数 a, b
5. 点击 "显示线性组合"
6. 观察绿色虚线向量 av₁+bv₂

**技巧：**
- 鼠标悬停图表查看坐标
- 使用工具栏缩放和平移
- 实时调整系数看变化

---

### 5️⃣ 特征值探索

**计算 2×2 矩阵特征值：**
1. 选择 "🎯 特征值探索"
2. 输入矩阵 [[4, 1], [2, 3]]
3. 点击 "计算特征值"
4. 查看结果: λ₁=5, λ₂=2
5. 点击 "3D 可视化特征向量"
6. 观察蓝色（原向量）和红色（变换后向量）

---

### 6️⃣ 矩阵分解

**LU 分解示例：**
1. 切换到 "🔬 矩阵分解"
2. 选择分解方法: LU 分解
3. 输入 3×3 矩阵
4. 点击 "执行分解"
5. 查看 L 和 U 矩阵
6. 查看逐步消元过程

---

### 7️⃣ 示例库

**快速加载经典例题：**
1. 点击 "📚 示例库"
2. 浏览三大类示例：
   - 经典矩阵类型
   - 特殊行列式
   - 典型例题
3. 点击任意卡片
4. 自动跳转到对应计算器并填充数据

**推荐尝试：**
- 3阶单位矩阵
- 范德蒙德行列式
- 求解线性方程组（经典三元一次）

## 💡 实用技巧

## ❓ 常见问题

### Q: 为什么有些结果显示为科学计数法？
A: math.js 对极小值（< 1e-10）自动显示为 0，极大值使用科学计数法。这是正常行为。

### Q: 如何清除当前计算？
A: 直接输入新数据并重新计算即可，结果会自动覆盖。

### Q: 能否保存计算历史？
A: 当前版本暂不支持。可以截图或使用浏览器书签保存重要结果。

### Q: 离线能用吗？
A: 首次加载需要网络（下载 CDN 资源），之后浏览器会缓存，可以离线使用。

### Q: 支持复数矩阵吗？
A: 当前版本仅支持实数矩阵。复数矩阵是未来扩展方向。
