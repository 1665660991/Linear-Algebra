# 🎬 线性代数学习软件 - 功能演示脚本

## 演示目标
在 5 分钟内展示软件的核心功能和教学价值

---

## 📋 演示流程

### 开场（30秒）
**话术：**
> "大家好！今天我要向大家介绍一款全新的线性代数学习工具。这是一款零依赖、跨平台的 Web 应用，只需浏览器即可运行。让我们看看它能做什么！"

**操作：**
- 打开浏览器
- 双击 `linear-algebra-calculator.html`
- 展示主界面

---

### 第一部分：矩阵计算器（1分钟）

**场景 1：计算行列式**
> "首先，我们来看最常用的功能——矩阵计算器。假设我们要计算这个 3×3 矩阵的行列式..."

**操作步骤：**
1. 点击 "🔢 矩阵计算器"
2. 输入矩阵：
   ```
   [1, 2, 3]
   [4, 5, 6]
   [7, 8, 9]
   ```
3. 选择 "行列式 det(A)"
4. 点击 "计算"
5. 展示结果：det(A) = 0

**亮点强调：**
> "注意看，下方还显示了专业的 LaTeX 公式排版，非常适合做笔记或报告。"

---

**场景 2：逐步模式**
> "对于初学者，我们提供了'逐步模式'，可以查看详细的计算过程..."

**操作步骤：**
1. 点击 "切换逐步模式"
2. 重新计算行列式
3. 展示下方的步骤说明

**教育价值：**
> "这样学生不仅能得到答案，还能理解算法是如何执行的。"

---

### 第二部分：行列式性质演示（1分钟）

**场景：交互式学习**
> "接下来，我们看看行列式的三大性质。传统教学只能口头讲解，但我们可以 interactive 演示！"

**操作步骤：**
1. 切换到 "📊 行列式专题"
2. 输入 2×2 矩阵 [[1, 2], [3, 4]]
3. 点击 "交换两行（变号）"
4. 展示：det 从 -2 变为 2

**强调：**
> "看到了吗？交换两行后，行列式的符号改变了！这种直观的演示比死记硬背效果好得多。"

**继续演示：**
- 点击 "某行乘k" → 展示行列式倍数变化
- 点击 "倍加变换" → 展示行列式不变

---

### 第三部分：方程组求解（1分钟）

**场景：高斯消元法**
> "线性方程组是线性代数的核心应用。让我们用高斯消元法求解这个三元一次方程组..."

**操作步骤：**
1. 切换到 "✏️ 方程组求解"
2. 输入增广矩阵：
   ```
   [2, 1, -1, 8]
   [-3, -1, 2, -11]
   [-2, 1, 2, -3]
   ```
3. 点击 "高斯消元法"
4. 展示解：x=2, y=3, z=-1
5. 滚动查看下方的详细步骤

**教学亮点：**
> "每一步都有详细说明：选主元、行交换、消元...学生可以跟着步骤自己动手验证。"

---

### 第四部分：向量可视化（1分钟）

**场景：线性组合**
> "向量空间的概念很抽象，我们用可视化工具让它变得直观！"

**操作步骤：**
1. 切换到 "📈 向量可视化"
2. 输入 v₁=(2,1), v₂=(1,3)
3. 点击 "绘制向量"
4. 展示蓝色和红色向量
5. 拖动滑块调整系数 a=1.5, b=0.5
6. 点击 "显示线性组合"
7. 展示绿色的 av₁+bv₂ 向量

**强调：**
> "通过实时调整系数，学生可以直观理解什么是'线性组合'，这是理解向量空间的基础！"

**交互演示：**
- 鼠标悬停图表查看坐标
- 使用工具栏缩放
- 快速调整系数看变化

---

### 第五部分：特征值与 3D 可视化（30秒）

**场景：几何意义**
> "特征值和特征向量的几何意义是什么？让我们用 3D 可视化来理解..."

**操作步骤：**
1. 切换到 "🎯 特征值探索"
2. 输入矩阵 [[4, 1], [2, 3]]
3. 点击 "计算特征值"
4. 展示：λ₁=5, λ₂=2
5. 点击 "3D 可视化特征向量"
6. 展示 Three.js 渲染的场景

**解释：**
> "蓝色箭头是原特征向量，红色箭头是变换后的向量。注意它们方向相同，只是长度被特征值缩放了！这就是特征向量的本质。"

---

### 第六部分：示例库（30秒）

**场景：快速上手**
> "如果你不知道从哪里开始，示例库是你的最佳选择！"

**操作步骤：**
1. 切换到 "📚 示例库"
2. 浏览三大类示例
3. 点击 "范德蒙德行列式"
4. 自动跳转到行列式计算器并填充数据
5. 点击计算查看结果

**强调：**
> "内置的经典例题覆盖了大一课程的核心知识点，一键加载，省时省力！"

---

### 结尾（30秒）

**总结优势：**
> "总结一下，这款工具的三大优势：
> 1. **零依赖**：无需安装，浏览器直接运行
> 2. **教学友好**：逐步展示 + 可视化
> 3. **功能完整**：7大模块覆盖核心知识点"

**呼吁行动：**

## 🎯 演示要点

### ✅ 要做的事
- 保持节奏流畅，每个功能 1 分钟左右
- 强调教育价值和实际应用场景
- 展示交互性（滑块、缩放、点击）
- 突出"零依赖"的便利性
- 用真实数据演示，避免枯燥的理论

### ❌ 避免的事
- 不要深入算法细节（时间有限）
- 不要展示所有功能（留有余地）
- 不要忽略错误处理（演示健壮性）
- 不要用太复杂的矩阵（保持简单）
- 不要跳过可视化部分（这是亮点）
