# 线性代数学习软件

## 📖 项目简介

这是一个面向大一学生的线性代数学习软件，采用纯静态 HTML + JavaScript 实现，**零依赖、跨平台**，只需浏览器即可运行。

## ✨ 核心功能

### 1. 🔢 矩阵计算器
- **基本运算**：加法、减法、数乘、矩阵乘法、矩阵幂
- **高级运算**：行列式、逆矩阵、转置、秩
- **特征分析**：特征值与特征向量计算（支持任意维度方阵）
- **逐步模式**：可切换即时计算或逐步展示算法过程
- **自定义维度**：支持 1×1 到 10×10 任意大小矩阵

### 2. 📊 行列式专题
- **Laplace 展开**：支持 n×n 行列式计算（自定义维度）
- **步骤展示**：详细展示余子式展开过程
- **性质演示**：交互式演示行列式的三大性质
  - 交换两行 → 变号
  - 某行乘 k → 行列式乘 k
  - 倍加变换 → 不变
- **特殊案例**：范德蒙德、三角行列式等

### 3. ✏️ 方程组求解器
- **高斯消元法**：完整步骤展示
- **克拉默法则**：适用于唯一解情况
- **解的判断**：唯一解/无穷多解/无解
- **几何可视化**：2D 图形展示方程交点

### 4. 📈 向量可视化
- **Plotly 交互图表**：支持缩放、平移
- **线性组合**：滑块实时调整系数 a, b
- **平行四边形法则**：直观展示向量加减
- **动态更新**：修改向量坐标即时刷新

### 5. 🎯 特征值探索
- **数值计算**：使用 math.js 精确求解
- **支持任意维度**：2×2 到 5×5 方阵
- **特征向量显示**：清晰列出所有特征值和对应特征向量

### 6. 🔬 矩阵分解实验室
- **LU 分解**：含逐步消元过程
- **QR 分解**：Gram-Schmidt 正交化
- **Cholesky 分解**：正定矩阵专用
- **SVD 分解**：奇异值分解

### 7. 📚 示例库
- **经典矩阵**：单位矩阵、对称矩阵、对角矩阵等
- **特殊行列式**：范德蒙德、三角行列式
- **典型例题**：教材常见习题（带完整解答）
- **一键加载**：点击示例自动填充到计算器

### 8. ✍️ 练习题系统（新增）
- **随机生成题目**：支持5种题型
  - 行列式计算
  - 矩阵运算（加减乘）
  - 求逆矩阵
  - 线性方程组求解
  - 特征值与特征向量
- **难度等级**：简单(2×2)、中等(3×3)、困难(4×4)
- **自动判题**：即时检查答案正确性
- **学习进度**：统计完成题目数和正确率
- **查看答案**：提供详细解题步骤

### 9. 📄 LaTeX导出（新增）
- **导出计算过程**：将所有计算结果转为LaTeX格式
- **选择性导出**：可选择导出哪些模块的内容
- **一键下载**：直接下载.tex文件
- **复制功能**：复制到剪贴板方便粘贴
- **标准格式**：包含完整的文档结构和数学公式包

## 🚀 使用方法

### 本地使用（推荐）
1. 直接双击 `linear-algebra-calculator.html` 文件
2. 在浏览器中打开即可使用
3. **无需安装任何软件，无需联网**（首次加载 CDN 资源后缓存）

### 在线部署
#### GitHub Pages
```bash
# 创建新仓库
git init linear-algebra-app
cd linear-algebra-app

# 添加文件
cp linear-algebra-calculator.html index.html
git add .
git commit -m "Initial commit"

# 推送到 GitHub
git remote add origin https://github.com/username/linear-algebra-app.git
git push -u origin main

# 在 GitHub Settings → Pages 启用
```

访问地址：`https://username.github.io/linear-algebra-app`

#### Netlify / Vercel
直接拖放 HTML 文件到部署平台，自动生成 HTTPS 链接

## 💡 使用技巧

1. **逐步学习模式**：
   - 点击"切换逐步模式"按钮
   - 每次计算都会显示详细推导步骤
   - 适合理解算法原理

2. **示例快速加载**：
   - 在"示例库"Tab 点击任意卡片
   - 自动跳转到对应计算器并填充数据
   - 节省手动输入时间

3. **向量交互**：
   - 拖动滑块实时查看线性组合变化
   - 鼠标悬停 Plotly 图表查看坐标
   - 使用工具栏缩放和平移

4. **矩阵维度调整**：
   - 先选择行列数
   - 点击"更新矩阵"按钮
   - 动态生成输入网格

## 🔧 常见问题

### Q: 为什么有些计算结果显示为科学计数法？
A: 这是 math.js 的默认行为。对于极小值（< 1e-10）会自动显示为 0。

### Q: 如何清除当前计算结果？
A: 重新点击"计算"按钮或用新数据覆盖。

### Q: 移动端触摸操作不流畅？
A: 建议使用桌面端获得最佳体验。移动端可使用双指缩放手势。

### Q: 能否保存计算历史？
A: 当前版本暂不支持。后续可扩展 localStorage 功能。
